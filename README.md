# Analyse-du-march-de-fitness-avec-Google-Trends

Dans ce projet, j'analyse des données Google Trends afin d'étudier l'évolution de la demande pour les entraînements sportifs et les solutions de fitness numériques.

L'objectif est d'identifier les tendances de recherche liées au fitness et d'en tirer des conclusions .

## Données utilisées

### workout.csv

Contient l'évolution mensuelle de la popularité mondiale du mot-clé **"workout"**.

**month**  Mois de l'observation |
**workout_worldwide** Indice de popularité Google Trends |

### three_keywords.csv
Contient la popularité mondiale de trois mots-clés :
- home workout
- gym workout
- home gym
### workout_geo.cs
Contient l'indice de popularité du mot-clé **"workout"** par pays sur la période 2018-2023
### three_keywords_geo.csv

Contient les indices de popularité par pays pour :

- home workout
- gym workout
- home gym
## Questions étudiées

### Question 1
À quelle période l'intérêt mondial pour le mot-clé **"workout"** a-t-il atteint son maximum ?

### Question 2
Quel mot-clé a été le plus populaire pendant la pandémie de Covid-19 ?

Quel est le mot-clé le plus populaire actuellement ?

### Question 3

Parmi les États-Unis, l'Australie et le Japon, quel pays présente l'intérêt le plus élevé pour le mot-clé **"workout"** ?
